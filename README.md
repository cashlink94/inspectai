# 🧠 InspectAI

### AI-Powered Steel Defect Detection

> Transforming industrial inspection with deep learning

---

## 🚀 Overview

**InspectAI** is an AI-driven visual inspection system that detects surface defects in steel with high precision.
It uses deep learning segmentation to identify, highlight, and quantify defects in real time.

Designed for **manufacturing, quality control, and maintenance teams**.

---

## ✨ Key Capabilities

* 🔍 **Automated Defect Detection**
  Identify surface defects instantly from uploaded images

* 🎯 **Multi-Class Segmentation**
  Detect up to 4 defect types with pixel-level precision

* 🎨 **Visual Intelligence**
  Color-coded overlays for intuitive analysis

* 📦 **Smart Localization**
  Bounding boxes highlight defect regions

* 📊 **Actionable Metrics**
  Defect coverage percentage for decision-making

---

## 🖼️ Product Experience

Upload a steel image → Get:

* ✅ Clean / Defective classification
